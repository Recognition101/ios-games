iOS Games Worth Playing

This is my list of games worth playing on iOS. Every game can be played offline, and none of these games offer in-app purchases that can be repeatedly purchased.

The games are grouped by features. Hovering over a particular game will show the full set of features that game supports. Here is the key describing what each feature icon means:

IconFeature
📱iPhone Support
💻iPad Support
🖥Apple TV Support
☁ Cloud Sync Support
🎮MFI Game Pad Support
🏁 Game has a defined end (not an arcade game)
🗺 Game has a "world" (as opposed to a menu of levels to choose from). Note: I include rogue-likes in this category.
